# Evaluasi Diri Jurnal

Aplikasi HTML mandiri untuk membantu pengelola jurnal melakukan evaluasi diri sebelum menyusun roadmap menuju jurnal internasional bereputasi. Aplikasi ini menyediakan kuesioner terstruktur untuk dua rumpun bidang, yaitu SosHum dan SainTek, kemudian menghasilkan rekomendasi prioritas roadmap berdasarkan skor kapasitas jurnal.

Halaman publik:

https://ikhwan-arief.github.io/evaluasi-diri-jurnal/

## Fitur Utama

- Kuesioner evaluasi diri untuk jurnal bidang SosHum dan SainTek.
- Skor kapasitas jurnal berbasis skala 0-4.
- Ringkasan status kesiapan jurnal.
- Daftar dimensi terlemah yang perlu diprioritaskan.
- Rekomendasi 5 prioritas roadmap utama.
- Rekomendasi kegiatan aktual, komponen biaya, mitra kerja, dan indikator terukur.
- Rencana kerja 12 bulan.
- Opsi salin hasil, cetak, dan simpan hasil sebagai file `.txt`.
- Dapat digunakan langsung di browser tanpa instalasi, server, akun, atau koneksi API.

## Panduan Ringkas Pemakaian

1. Buka halaman publik aplikasi:
   https://ikhwan-arief.github.io/evaluasi-diri-jurnal/
2. Isi nama jurnal dan tanggal evaluasi.
3. Pilih bidang kajian: `SosHum` atau `SainTek`.
4. Jawab semua pertanyaan dengan skala 0-4 sesuai kondisi nyata jurnal.
5. Klik `Hitung Rekomendasi`.
6. Baca ringkasan skor, dimensi terlemah, prioritas roadmap, dan rencana 12 bulan.
7. Gunakan tombol `Salin Hasil`, `Simpan Hasil (.txt)`, atau `Cetak` untuk mendokumentasikan hasil evaluasi.

## Skala Penilaian

- `0`: Belum tersedia atau belum ada bukti.
- `1`: Ada secara informal dan belum stabil.
- `2`: Terdokumentasi sebagian, tetapi belum konsisten.
- `3`: Berjalan konsisten dengan penanggung jawab jelas.
- `4`: Terukur, dievaluasi, dan menunjukkan perbaikan.

## Catatan Privasi

Aplikasi ini berjalan sepenuhnya di browser. Data yang diisikan tidak dikirim ke server, tidak memakai basis data, dan tidak memakai layanan LLM/API. File hasil hanya dibuat ketika pengguna menekan tombol `Simpan Hasil (.txt)`.
